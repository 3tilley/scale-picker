# scale-picker

A randomised CAGED scale picker for guitarists — runs in the browser with no build required.

🎸 **[Open Scale Picker on GitHub Pages](https://3tilley.github.io/scale-picker/)**

## Features

- Randomise root note, scale quality (Major/Minor), CAGED position, direction, register, and scale order
- Selected CAGED letter is highlighted larger in amber, e.g. **C A G E D** with the chosen one enlarged
- Play the scale as eighth notes via the browser Web Audio API (configurable BPM)
- Spacebar shortcut to instantly generate a new scale
- Mobile-friendly: settings hidden behind a hamburger drawer, large tap area to generate

## GitHub Pages

This is a pure static site (HTML + CSS + JS, no build step). Once GitHub Pages is enabled on the repository (Settings → Pages → Deploy from branch `main`, root `/`), it is automatically available at:

```
https://3tilley.github.io/scale-picker/
```
